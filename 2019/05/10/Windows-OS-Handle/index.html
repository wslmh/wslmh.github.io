获取进程句柄（HANDLE）和窗口句柄（HWND）

当在一台主机中时，可以根据进程的ID获取进程句柄
也可以根据进程名称来获取进程句柄，

获取窗口句柄：

当想获取窗口句柄时，可以使用窗口的类名或窗口的标题来获取窗口句柄

进程句柄和窗口句柄的区别：

窗口句柄和进程句柄是不一样的，一个进程是可以有多个窗口句柄的，通过窗口句柄可以得到进程句柄，返过来就比较麻烦了，因为一个进程可以有多个窗口，但是得到父窗口还是比较容易的。

据窗口句柄得到进程句柄：

如何查找到当前窗口所在进程的相关数据，如进程句柄、进程
首先通过类名 或 窗体的名字 得到映射窗口的窗口句柄 windows提供的API函数 FindWindow（）函数就能得到这个句柄HWND FindWondow（lpclassname,lpwindowsname）
然后通过得到的窗口句柄得到窗口所在进程的ID
GetWindowThreadID（窗口句柄，所在进程的ID）
然后再通过进程的ID得到进程的进程句柄
句柄=openProcess（，，句柄）；

HANDLE 和 HWND 的区别:

HWND是HANDLE的一种，但是HANDLE并不是HWND.
HANDLE是操作系统一系列内核对象的句柄。
HWND仅是窗口对象的句柄
