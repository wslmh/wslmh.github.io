Hexo + Github Page 建立个人博客

Hexo 与 Github Page配合，建立静态托管网站作为个人博客。其中，Hexo使用的是NexT主题。本文重点记录建立网站后的个性化配置，方便重建参考。

首先是创建个人的Github Page，这个在网络上有很多，可以搜索得到。

下载安装Hexo，在安装Hexo之前，需要先安装Git和Node.js。之后在目标文件夹中安装Hexo即可，具体安装方式参考网络。

Hexo安装完成后，使用 hexo init 命令进行初始化，初始化完成后，再分别使用 hexo g 和 hexo s 命令生成、运行hexo。

在本地浏览器打开 http://localhost:4000/ 查看是否正常显示。

配置目录下 _config.yml 文件，设置目标github网站和配置，使用 hexo d 上传更新，等待一会后即可从静态地址中打开博客网站。
